XII. Les formulaires

Les classes de Bootstrap enrichissent les formulaires du html.

Ceux-ci sont plus visibles et moins austères.

La classe .form-group

Elle permet de grouper un input et son label.On utilise le tag label pour décrire l'input. Dans son attribut for du laebl on affecte la valeur de l'id de l'input.Visuellement, le label se situe au-dessus de l'input. On peut ajouter un attribut placeholder pour mettre une valeur qui ne sera pas prise en compte: nom@exemple.com. Qui est donné à titre d'exemple.
