Les boutons Bootstrap

Bootstrap fournit une classe .btn qui doit être combinée avec d'autres que nous allons voir.

La classe .btn et suivit d'une classe de couleur

Pour cela on ajoute un suffixe de couleur définit dans Bootstrap.

Les classes .btn .btn-primary

Bouton de couleur bleue

Les classes .btn .btn-secondary

Bouton de couleur grise

Les classes .btn .btn-success

Bouton de couleur verte

Les classes .btn .btn-danger

Bouton de couleur rouge
