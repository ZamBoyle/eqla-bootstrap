Exercice n°1 - Insertion des éléments CSS et JS

  1. Créez une nouvelle page html dans VsCode.
  2. Ajoutez la feuille de style de Bootstrap qui se trouve dans le répertoire Styles :
    3. bootstrap.min.css
  3. Ajoutez les fichiers JS, qui se trouvent dans le répertoire JS, à la fin de la page juste avant la fermeture de la base body :
    4. jquery-3.5.1.slim.min.js Bootstrap.bundle.min.js
  4. Créer votre première page Bootstrap en incluant un div avec la classe .container.
  5. Ajoutez dans le div précédent ce texte « Hello World, from Bootstrap » en titre1 (h1).
  6. Enregistrez sous votre fichier avec le nom Exercice1.html

Exercice n°8.2 - Le composant carousel - Le contrôle Précédent

  1. Utilisez le précédent carousel
  2. Ajoutez un id avec comme valeur monCarousel au div ayant la classe carousel.
  3. Positionnez-vous après la fermeture du div ayant la classe carousel-inner.
  4. Ouvrez une balise a et définissez les attributs suivants :
    1. class="carousel-control-prev"
    2. href="#monCarousel" (C’est l’id du carousel définit au point 3)
    3. role="button"
    4. data-slide="prev"
    5. Ajoutez <span class="carousel-control-prev-icon" aria-hidden="true"></span>
    6. Fermez la balise a

Exercice n°8.5 - Le composant carousel - Les contrôles pause/play

Cet exercice sort un peu du cadre du cours. Mais comme vous me l'avez demandé, vous pouvez voir sa solution dans la correction des exercices.
  1. Ouvrez dans VsCode la page Template.html
  2. Allez à la ligne n°18.
  3. Copiez le code html du carousel de la correction de l'exercice 8.5
  4. Creez un nouveau fichier JS
  5. Dans ce fichier JS, collé le code JS de la correction de l'exercice 8.5: Vous avez le choix entre du JS basé sur jQuery ou en pur Javascript.
  6. Revenez dans le fichier html et faites une référence au fichier js que vous venez de créer.
  7. Testez ce carousel avec les controles pause/play.
  8. Je vais vous expliquer comment tout cela fonctionne. :)

Exercice n°10 - Les couleurs

    • Couleur de texte: rouge (danger), texte:"Je suis en rouge".

      Couleur de texte: blanc, couleur de fond: rouge, texte:"Je suis en blanc sur fond rouge.
      Couleur de texte: blanc, couleur de fond: dark, texte:"Je suis en blanc sur fond dark.
  6. Ajoutez un fond de couleur (light) à votre div class="container".
  7. Faites un div ayant du texte:
      • Couleur de texte: blanc, couleur de fond: dark, texte:"Je suis en blanc sur fond dark.

Exercice n°12 - Les tableaux

  1. Ouvrez dans VsCode la page Template.html
  2. Allez à la ligne n°18.
  3. Ajoutez un div class="container".
  4. Dans le précédent div, insérez le tableau suivant
    Cliquer pour copier le tableau 
    <table>
    <thead>
        <tr>
            <th scope="col">Nom</th>
            <th scope="col">Prénom</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Flower</td>
            <td>Margueritte</td>
        </tr>
        <tr>
            <td>Thumas</td>
            <td>Marie</td>
        </tr>
        <tr>
            <td>Ange</td>
            <td>Gabriel</td>
        </tr>
    </tbody>
</table>
  5. Ajoutez à la balise table la classe .table pour le styliser à la sauce Bootstrap.
  6. Ajoutez à la balise table la classe .table-striped pour le zébrer.
  7. Ajoutez à la balise table la classe .table-bordered pour ajouter une bordure.
  8. Ajoutez à la balise thead la classe .thead-dark pour rendre nos entêtes de couleur noire.
