Exercices Bootstrap

Exercice n°1

  1. Créez une nouvelle page html dans VsCode.
  2. Ajoutez la feuille de style de Bootstrap qui se trouve dans le répertoire Styles :
    3. bootstrap.min.css
  3. Ajoutez les fichiers JS, qui se trouvent dans le répertoire JS, à la fin de la page juste avant la fermeture de la base body :
    4. jquery-3.5.1.slim.min.js Bootstrap.bundle.min.js
  4. Créer votre première page Bootstrap en incluant un div avec la classe .container.
  5. Ajoutez dans le div précédent ce texte « Hello World, from Bootstrap » en titre1 (h1).
  6. Enregistrez sous votre fichier avec le nom Exercice1.html

Exercice n°3

  2. Allez à la ligne n°18.

Exercice n°4

  1. Ouvrez dans VsCode la page Template.html
  2. Allez à la ligne n°18.
  3. Dans un paragraphe, mettez ce texte en minuscules : « JE SUIS UN TEXTE EN MINUSCULES ».
  4. Dans un paragraphe, mettez ce texte en majuscules : « je suis un texte en minuscules ».
  5. Dans un paragraphe, placez à droite le texte : « Je suis placé à droite de l’écran ».
  6. Dans un paragraphe, placez à droite le texte : « Je suis placé à droite de l’écran » si la résolution est d’au moins lg.
  7. Enregistrez sous votre fichier avec le nom Exercice4.html

Exercice n°5

  1. Téléchargez une grande image sur Internet ou utilisez une que vous avez déjà.
  2. Ouvrez dans VsCode la page Template.html
  3. Allez à la ligne n°18.
  4. Ajoutez cette image sans qu’elle soit responsive.
  5. Ajoutez cette image pour qu’elle soit responsive.
  6. Ajoutez cette image et fais-en une vignette responsive.
  7. Enregistrez sous votre fichier avec le nom Exercice5.html

Exercice n°7
